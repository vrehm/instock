Get a list of all blogs

<blogs type="array">
  <blog>
    <commentable>no</commentable>
    <created-at type="datetime">2009-06-24T11:53:12-04:00</created-at>
    <feedburner nil="true"></feedburner>
    <feedburner-location nil="true"></feedburner-location>
    <handle>bigcheese-blog</handle>
    <id type="integer">1314995009</id>
    <title>Mah Blog</title>
    <updated-at type="datetime">2006-02-01T19:00:00-05:00</updated-at>
  </blog>
</blogs>

Creating a new Blog

Create a new blog

POST /admin/blogs.xml

Create a new empty blog

Request

           
<?xml version="1.0" encoding="UTF-8"?>
<blog>
  <title>Apple main blog</title>
</blog>

Response

           
HTTP/1.1 201 Created
<?xml version="1.0" encoding="UTF-8"?>
<blog>
  <commentable>no</commentable>
  <created-at type="datetime">2009-06-24T11:53:23-04:00</created-at>
  <feedburner nil="true"></feedburner>
  <feedburner-location nil="true"></feedburner-location>
  <handle>apple-main-blog</handle>
  <id type="integer">1314995043</id>
  <title>Apple main blog</title>
  <updated-at type="datetime">2009-06-24T11:53:23-04:00</updated-at>
</blog>

Modifying an existing Blog

Update a blog

PUT /admin/blogs/#{id}.xml

Update an existing blog title

Request

           
<?xml version="1.0" encoding="UTF-8"?>
<blog>
  <title>IPod Updates</title>
  <id type="integer">1314995009</id>
</blog>

Response

           
HTTP/1.1 200 OK
<?xml version="1.0" encoding="UTF-8"?>
<blog>
  <commentable>no</commentable>
  <created-at type="datetime">2009-06-24T11:53:12-04:00</created-at>
  <feedburner nil="true"></feedburner>
  <feedburner-location nil="true"></feedburner-location>
  <handle>bigcheese-blog</handle>
  <id type="integer">1314995009</id>
  <title>IPod Updates</title>
  <updated-at type="datetime">2009-06-24T11:53:24-04:00</updated-at>
</blog>
PUT /admin/blogs/#{id}.xml

Update an existing blog title and handle and also activate comments

Request

           
<?xml version="1.0" encoding="UTF-8"?>
<blog>
  <handle>ipod-updates</handle>
  <title>IPod Updates</title>
  <id type="integer">1314995009</id>
  <commentable>moderate</commentable>
</blog>

Response

           
HTTP/1.1 200 OK
<?xml version="1.0" encoding="UTF-8"?>
<blog>
  <commentable>moderate</commentable>
  <created-at type="datetime">2009-06-24T11:53:12-04:00</created-at>
  <feedburner nil="true"></feedburner>
  <feedburner-location nil="true"></feedburner-location>
  <handle>ipod-updates</handle>
  <id type="integer">1314995009</id>
  <title>IPod Updates</title>
  <updated-at type="datetime">2009-06-24T11:53:25-04:00</updated-at>
</blog>
