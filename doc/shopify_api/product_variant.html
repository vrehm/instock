Creating a new Product Variant

Create a new product variant

POST /admin/products/#{id}/variants.xml

Create a new product variant

Request

           
<?xml version="1.0" encoding="UTF-8"?>
<variant>
  <price>1.00</price>
  <option1>Default</option1>
</variant>
