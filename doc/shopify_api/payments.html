Shopify API - Payments

Receive a list of all Payments

GET /admin/orders/#{id}/payments.xml

Get the XML Representation of all money transfers on a given order.

Response

           
HTTP/1.1 200 OK
<?xml version="1.0" encoding="UTF-8"?>
<authorizations type="array">
  <authorization>
    <amount type="decimal">199.00</amount>
    <authorization>authorization-key</authorization>
    <capture type="boolean">false</capture>
    <created-at type="datetime">2005-08-01T11:57:11-04:00</created-at>
    <order-id type="integer">725106799</order-id>
    <status>success</status>
    <kind>Authorization</kind>
    <receipt>testcasetrue</receipt>
  </authorization>
</authorizations>

Receive a single Payments

GET /admin/orders/#{id}/payments/#{id}.xml

Get the XML Representation of a specific payment.

Response

           
HTTP/1.1 200 OK
<?xml version="1.0" encoding="UTF-8"?>
<authorization>
  <amount type="decimal">199.00</amount>
  <authorization>authorization-key</authorization>
  <capture type="boolean">false</capture>
  <created-at type="datetime">2005-08-01T11:57:11-04:00</created-at>
  <order-id type="integer">725106799</order-id>
  <status>success</status>
  <kind>Authorization</kind>
  <receipt>testcasetrue</receipt>
</authorization>
